# Node.js Hello World 서버

이 프로젝트는 간단한 "Hello, World!"를 응답하는 기본적인 Node.js HTTP 서버입니다.

## 실행 방법 (Node.js)

이 서버를 실행하기 위해 외부 라이브러리 설치가 필요 없습니다. Node.js가 설치되어 있다면 다음 명령어를 사용하여 바로 서버를 시작할 수 있습니다.

### 1. 서버 시작

아래 명령어를 터미널에서 실행하여 주십시오.

```bash
node server.js
```

### 2. 서버 확인

서버가 정상적으로 시작되면 터미널에 `server listening on port 8080` 메시지가 출력됩니다.

웹 브라우저나 `curl`을 사용하여 `http://localhost:8080`으로 접속하면 아래와 같은 응답을 확인할 수 있습니다.

```
Hello, World!
Hostname: [실행 중인 컴퓨터의 호스트 이름]
```

### 3. 서버 종료

터미널에서 `Ctrl + C`를 눌러 서버를 종료할 수 있습니다.

---

## 실행 방법 (Docker)

Docker가 설치되어 있다면, 아래의 명령어를 사용하여 컨테이너 환경에서 서버를 실행할 수 있습니다.

### 1. Docker 이미지 빌드

프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 `my-node:hello` 이미지를 빌드합니다.

```bash
docker build -t my-node:hello .
```

### 2. Docker 컨테이너 실행

빌드된 이미지를 사용하여 컨테이너를 실행합니다.
- `-p` 옵션은 호스트의 8080 포트를 컨테이너의 8080 포트와 매핑합니다.
- `-d` 옵션은 컨테이너를 백그라운드에서 실행합니다.
- `--rm` 옵션은 컨테이너가 중지될 때 자동으로 삭제되도록 합니다.

```bash
docker run --rm -p 8080:8080 -d my-node:hello
```

### 3. 서버 확인

웹 브라우저나 `curl`을 사용하여 `http://localhost:8080`으로 접속하면 동일한 응답을 확인할 수 있습니다.

```
Hello, World!
Hostname: [컨테이너의 호스트 이름]
```

### 4. 실행 중인 컨테이너 확인 및 종료

아래 명령어로 실행 중인 컨테이너의 ID를 확인할 수 있습니다.

```bash
docker ps
```

확인한 ID를 사용하여 컨테이너를 중지할 수 있습니다. (`--rm` 옵션으로 인해 컨테이너는 중지와 동시에 삭제됩니다.)

```bash
docker stop [컨테이너 ID]
```
